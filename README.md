# 🩺 Maya Scene Doctor AI (V3)

![Maya](https://img.shields.io/badge/Maya-2022%2B-blue)

**An AI assistant that lives inside Maya.** It automatically scans your scene, explains what's wrong and why, suggests fixes with runnable code, and can even verify its own work — all without leaving Maya.

![Screenshot](images/screenshot.png)

---

## 🚀 Quick Start

1. **Download** the files and place them in:  
   `Documents/maya/scripts/`
2. In the **Maya Script Editor** (Python), run:
```python
import main
main.show()
```

---

## ✨ Key Features

- **Multi-Agent Logic**: Specialized agents for **Analysis**, **Code Generation**, and **Visual Inspection**.
- **Zero Setup**: Uses native Maya libraries. No `pip install` required.
- **Smart Fixes**: Generates and runs Python code directly in your scene.
- **Search Integration**: Look up tutorials and tools on Gumroad/GitHub directly from the chat.
- **Flexible AI**: Connect to **Ollama** (Local) or any **API** (Groq, GPT-4, etc.).
